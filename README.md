## Curriculum: 3 Days x 3 Hours (9 Hours Total)

### Day 1 — Docker Fundamentals & Images

**Session 1: Containers & Docker Setup (3h)**
- Containers vs VMs, Docker architecture
- Installing Docker Engine + Docker Compose on Ubuntu
- Docker CLI basics: run, ps, logs, exec, stop, rm
- Writing Dockerfiles, building custom images
- **Lab:** Install Docker on a fresh VPS, build and run a custom image

### Day 2 — Networking, Volumes & Domain/SSL

**Session 2: Compose, Volumes & Production Access (3h)**
- Bind mounts vs named volumes, data persistence
- Docker networks, multi-container orchestration with Compose
- Domain pointing + Nginx reverse proxy + Let's Encrypt SSL for containerized apps
- **Lab:** Deploy a multi-container app (app + database) with a real domain and HTTPS

### Day 3 — Security, Monitoring & Freelance Readiness

**Session 3: Hardening, Monitoring & Client Acquisition (3h)**
- UFW firewall with Docker, non-root containers, secrets basics
- Container monitoring (docker stats, healthchecks, log rotation)
- Client project simulation: full deploy → troubleshoot → handover documentation
- Freelance readiness: Upwork/Freelancer/Guru proposal writing, pricing, LinkedIn outreach for direct clients
- **Final Assessment:** Submit 1 Upwork proposal + 1 LinkedIn outreach message (reviewed live)
